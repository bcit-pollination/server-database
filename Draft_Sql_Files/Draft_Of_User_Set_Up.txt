GROUP 1 (Read only controls):
-They can SELECT.

GROUP 2 (User level controls):
-Can use queries for updating and insertion.
-They can't have direct access to INSERT and UPDATE
-They are not allowed to DELETE.
-They can SELECT.

GROUP 3 (Administrator):
-Not allowed to drop tables or database schemas
-Not allowed superuser commands
-We are allowed to use SELECT, UPDATE, DELETE
-We're allowed to INSERT
-We can create queries and Grant users permissions to those queries
-We can't create users.
-We can SELECT.
